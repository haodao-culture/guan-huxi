# 靜心・觀呼吸

《關於淨念、靜心「觀」呼吸》線上閱讀網站，系統化呈現觀呼吸的內修心法與練習——從一次觀呼吸練習，到捨識用根、自我止息。並附觀呼吸計時器與百日基本功打卡，陪伴日常練習。

🔗 線上閱讀：<https://haodao-culture.github.io/guan-huxi/>

## 內容原則

- 全程以原文呈現，網站只做章節編排、導覽與練習工具，**不改寫、不加註解**。
- 計時器依原文「不數息、不調節呼吸，讓身體自然呼吸」設計，只做安靜的時間陪伴。
- 百日打卡紀錄僅存於使用者本機（localStorage），不會上傳。

## 檔案結構

```
index.html          # 著陸頁：主視覺 + 觀呼吸計時器 + 百日基本功 + 章節目錄
ch01.html … ch11.html   # 四部十一章逐章閱讀
breathing.css       # 樣式
breathing.js        # 計時器與百日打卡
favicon.svg
```

## 本地預覽

```bash
python3 -m http.server 8000
```

然後開啟 <http://localhost:8000>。

## 部署

推到 `main` 分支即由 GitHub Pages 自動發布。
